# 🎬 Movie Bucket List

Check it out here! https://movie-bucketlist.vercel.app/

A personal movie bucket list tracker built for discovering, organizing, and exploring some of the greatest films ever made.
This project started with the Top 100 IMDb movies powered through the [OMDb API](https://www.omdbapi.com/), then expanded into a customizable collection of additional films that sparked personal interest and curiosity.

## ✨ Features

### 🎞️ Curated Movie Collection

- Browse the Top 100 IMDb movies
- Add additional movies that captivate your interest
- Organized movie tracking experience
- Movie details pulled dynamically from the OMDb API
- Editing and management controls are gated to authorized users (project owner/admin access)

### 🎡 Wheel Picker

- Interactive wheel picker for choosing what movie to watch next
- Helps eliminate decision fatigue
- Fun randomized movie selection experience

## 🧠 Why I Built This

I wanted a personal movie tracker that felt more interactive and visually engaging than a standard list.

The goal was to combine:

- movie discovery
- bucket list tracking
- randomized selection
- and a polished responsive UI

into a single experience built around films I genuinely want to watch.
